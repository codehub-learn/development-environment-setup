# Hardware and software requirements for Java Standard Edition development (Java SE 21 or later)
&nbsp;
&nbsp;


## Development IDE
IntelliJ IDEA Community Edition is the recommended IDE (if you already have the Ultimate edition, you do not need to do anything else). 

Download IntelliJ IDEA **[IntelliJ for all operating systems](https://www.jetbrains.com/idea/download/?section=windows)** and install. This will start with the Ultimate Edition (paid), and after 30 days it will switch to the Community Edition. This is fine, we only need the Community Edition. If you prefer a different IDE, that is also fine as the examples do not make use of any IDE specific functionality. Support in the training, however, will only be provided for IntelliJ.

## Java Development Tools
1. If you do not have Java 21 or later, download and install Java 25 from this location **[Latest Java JDK downloads](https://adoptium.net/en-GB/temurin/releases)**. Earlier versions of Java are out of support and not recommended to use, but if you already have them on you machine, you do not need to uninstall them.

## Other tools
1. Download and install **[Git Version Control](https://git-scm.com/downloads)** (free software). If you have an older version of git, it is highly recommended that you update to the latest version.
2. Create a **[Github account](https://github.com/join)** if you don't already have one (free to use).

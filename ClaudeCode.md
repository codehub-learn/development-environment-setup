# Claude Code Training: Preparation & Prerequisites

Welcome to the **Claude Code** training with **Python** examples. To ensure a smooth hands-on experience, please install the required software and create the necessary accounts before the session starts.

---

## 🛠️ Required Software

Below you will find the minimum versions and download links for the software:

| Software | Minimum Version | Download Link |
| :--- | :---: | :--- |
| **Node.js** | `v18+` | [Node.js Downloads](https://nodejs.org/en/download) |
| **Python** | `v3.10+` | [Python Downloads](https://www.python.org/downloads/) |
| **uv** *(Python Package Manager)* | `v0.1+` | [uv Installation Guide](https://docs.astral.sh/uv/getting-started/installation/) |
| **Git** *(Version Control)* | `v2.35+` | [Git Downloads](https://git-scm.com/downloads) |
| **Docker** | Latest | [Docker Desktop](https://www.docker.com/products/docker-desktop/) |
| **VS Code** *(Recommended IDE)* | Latest | [VS Code Downloads](https://code.visualstudio.com/) |
| **Claude Code** | Latest | [Claude Code Guide](https://support.anthropic.com/) |

> 💡 **Note for Claude Code:** Claude Code is installed via your terminal (after installing Node.js) by running the following command:  
> `npm install -g @anthropic-ai/claude-code`

---

## 👤 Accounts & Subscriptions

To use Claude Code and manage your code repositories, the following are required:

1. **Anthropic Account & Claude Subscription**
   * Create an account at [Anthropic](https://claude.ai/).
   * An active **Claude Pro** subscription or access via the **Anthropic API Console** (with a valid payment method/credits) is required, as Claude Code consumes tokens during usage.
2. **GitHub Account**
   * Create a free account at [GitHub Sign Up](https://github.com/join).
   * This will be used for version control and managing our Python projects.

---
*Please complete these installations and registrations beforehand so we can jump straight into the practical exercises!*

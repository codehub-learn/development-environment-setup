# Installation Instructions for Trainees

Below are the required tools for the training program.  
Each section includes the download link and a few helpful hints.

---

## Page 1 — JDK (Java Development Kit)

**Download:** https://adoptium.net/temurin

- Choose **Temurin 17** or **Temurin 21** (both are LTS and stable).
- Download the installer for your operating system.
- After installation, verify with: `java -version`.
- IntelliJ will automatically detect the JDK on first launch.

---

## Page 2 — IntelliJ IDEA Community Edition

**Download:** https://www.jetbrains.com/idea/download

- Select **Community Edition** (free).
- Enable **“Add to PATH”** during installation if available.
- On first launch, choose **“Do not import settings”**.
- JUnit, TestNG, Selenium, and Maven dependencies will be added later inside IntelliJ.

---

## Page 3 — Git

**Download:** https://git-scm.com/downloads

- Install using the default settings.
- Verify installation with: `git --version`.
- IntelliJ will automatically detect Git for version control features.
- Used for cloning, committing, branching, and pushing to GitHub.

---

## Page 4 — GitHub Account

**Create Account:** https://github.com

- Use a personal email address you can easily access.
- Enable **two-factor authentication** for security (recommended).
- You will use GitHub to store and share your projects.
- Remember your username for Git configuration.

---

## Page 5 — Postman

**Download:** https://www.postman.com/downloads

- Install the **Desktop App** (not the web version).
- Sign in with Google or create a free Postman account.
- Used for sending API requests and testing REST endpoints.
- No paid features are required for this course.

---

## Page 6 — Google Chrome

**Download:** https://www.google.com/chrome

- Recommended browser for UI testing and Selenium automation.
- Chrome DevTools will be used for inspecting elements and debugging.
- Keep Chrome updated to avoid WebDriver compatibility issues.

---

## Page 7 — ChromeDriver (Optional)

**Download:** https://chromedriver.chromium.org/downloads

- Only needed if not using WebDriverManager in Java.
- Download the version that matches your installed Chrome version.
- Extract the file somewhere easy to find.
- If using WebDriverManager, this step can be skipped.

---
